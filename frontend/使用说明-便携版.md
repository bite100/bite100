# 使用便携版（无需安装包）

## ✅ 当前状态

Electron 应用已经成功打包到 `release/win-unpacked/` 目录！

## 📦 使用方法

### 方法 1：直接运行（推荐）

1. **找到可执行文件**：
   - 路径：`frontend/release/win-unpacked/P2P 交易所.exe`
   - 如果找不到，检查 `win-unpacked` 目录中的所有文件

2. **双击运行**：
   - 直接双击 `P2P 交易所.exe` 即可启动应用
   - 无需安装，无需管理员权限

3. **分发**：
   - 可以将整个 `win-unpacked` 文件夹复制到任何位置
   - 或压缩成 ZIP 文件分发

### 方法 2：创建快捷方式

1. 右键点击 `P2P 交易所.exe`
2. 选择"创建快捷方式"
3. 将快捷方式放到桌面或开始菜单

## 🔍 查找可执行文件

如果找不到 `P2P 交易所.exe`，请检查：

```powershell
cd d:\P2P\frontend\release\win-unpacked
Get-ChildItem -Recurse -Filter *.exe
```

## 📋 目录结构

```
win-unpacked/
├── P2P 交易所.exe          # 主程序
├── resources/               # 应用资源
│   └── app.asar            # 打包的应用代码
├── locales/                # 语言文件
└── ...                     # Electron 运行时文件
```

## ⚠️ 注意事项

1. **首次运行**：Windows 可能显示"未知发布者"警告，点击"仍要运行"即可
2. **文件位置**：不要删除 `win-unpacked` 目录中的任何文件
3. **更新**：更新时需要替换整个 `win-unpacked` 目录

## 🚀 创建安装包（需要管理员权限）

如果需要创建安装包，请：

1. **以管理员身份打开 PowerShell**
2. **执行**：
   ```powershell
   cd d:\P2P\frontend
   $env:CSC_IDENTITY_AUTO_DISCOVERY='false'
   npm run electron:build
   ```

或者使用 GitHub Actions 自动构建。

## 💡 推荐

**便携版已经可以使用了！** 无需等待安装包构建完成。
